<objective>
Create the permissions guidance subsystem for Transy's Accessibility requirement.

Purpose: When the global key monitor cannot start because Accessibility permission is missing, show a
small, matter-of-fact guidance window — not a system alert — telling the user exactly what to do.
The window must re-appear on every failed trigger attempt rather than staying silent after the first dismissal,
and it must notify the app when permission becomes available so monitoring can start without relaunch.

Output: `Transy/Permissions/GuidanceView.swift` (SwiftUI content) and
`Transy/Permissions/GuidanceWindowController.swift` (AppKit host, singleton, AX-gated, trust-polling).
</objective>

<task type="auto">
  <name>Task 2: GuidanceWindowController — AppKit singleton host with AX gate</name>
  <files>Transy/Permissions/GuidanceWindowController.swift</files>
  <action>
Create `Transy/Permissions/GuidanceWindowController.swift`. This is an `@MainActor` `NSWindowController`
singleton that wraps `GuidanceView` in an `NSHostingController`, checks `AXIsProcessTrusted()` before
showing the window, and emits a callback after a lightweight poll observes that permission has been granted.

Critical implementation rules (from CONTEXT.md locked decisions + RESEARCH.md patterns):

1. **Singleton pattern:** `static let shared = GuidanceWindowController()` with private init.
2. **AX gate in showIfNeeded():** Must check `AXIsProcessTrusted()` first. If trusted, return immediately
   (silent no-op). Do NOT use `AXIsProcessTrustedWithOptions(prompt:true)` — the system prompt is generic
   and bypasses this custom guidance window.
3. **Re-entrant on every call:** Do NOT track "already shown" state. Every call to `showIfNeeded()` when
   `!AXIsProcessTrusted()` must bring the window to front. This is the locked decision: "show guidance
   again on each failed trigger attempt."
4. **NSApp.activate() before makeKeyAndOrderFront:** Required for LSUIElement apps to surface a window
   (same pattern as Phase 1's Settings window).
5. **Lazy window construction:** Build the NSWindow on first call, reuse on subsequent calls.
6. **Window style:** `.titled, .closable` only — no resizing, no miniaturize.
7. **Window level:** `.floating` so it appears above other apps' windows.
8. **Import:** `import ApplicationServices` for `AXIsProcessTrusted()`.
9. **Trust poll:** After the window is shown, start a lightweight `Timer` polling `AXIsProcessTrusted()`
   every ~2 seconds. When permission becomes true, invalidate the timer, hide the window, and call
   `onPermissionGranted`.
10. **No duplicate timers:** Repeated `showIfNeeded()` calls while the guidance window is already visible
    must not create multiple timers.

```swift
import AppKit
import ApplicationServices
import SwiftUI

@MainActor
final class GuidanceWindowController: NSWindowController {

    static let shared = GuidanceWindowController()
    var onPermissionGranted: (() -> Void)?

    private var trustPollTimer: Timer?

    private init() {
        super.init(window: nil)
    }

    required init?(coder: NSCoder) { fatalError("not used") }

    func showIfNeeded() {
        guard !AXIsProcessTrusted() else { return }

        if window == nil {
            let hosting = NSHostingController(rootView: GuidanceView())
            hosting.view.frame = NSRect(x: 0, y: 0, width: 340, height: 1)
            // Let the hosting controller size the window to fit content
            let win = NSWindow(contentViewController: hosting)
            win.title = "Transy — Accessibility Access"
            win.styleMask = [.titled, .closable]
            win.level = .floating
            win.isReleasedWhenClosed = false   // retain for reuse
            self.window = win
        }

        startTrustPolling()
        NSApp.activate()
        window?.makeKeyAndOrderFront(nil)
    }

    private func startTrustPolling() {
        guard trustPollTimer == nil else { return }

        trustPollTimer = Timer.scheduledTimer(withTimeInterval: 2.0, repeats: true) { [weak self] timer in
            MainActor.assumeIsolated {
                guard AXIsProcessTrusted() else { return }
                timer.invalidate()
                self?.trustPollTimer = nil
                self?.window?.orderOut(nil)
                self?.onPermissionGranted?()
            }
        }
    }
}
```

Note: `hosting.view.frame` is set to a minimal height; NSHostingController auto-sizes to the SwiftUI
content. Do not hardcode a specific height — let SwiftUI determine it.
