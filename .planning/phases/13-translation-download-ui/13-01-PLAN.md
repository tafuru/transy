<objective>
Replace the manual "Open Language & Region" model-download guidance with the Translation framework's built-in automatic download prompt. Remove the `.missingModel` preflight short-circuit so `session.translate()` proceeds when a model is supported but not installed — the framework automatically prompts the user to download it.

Purpose: TDL-01 — Users no longer need to manually navigate System Settings to install translation models.
Output: Cleaner translation flow with framework-native download UX; ~200 lines of guidance infrastructure deleted.
</objective>
