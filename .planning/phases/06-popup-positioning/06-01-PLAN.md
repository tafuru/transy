    Modify PopupController to use PopupPositionCalculator for cursor-proximate placement:

    1. **Add stored properties** (after the existing `onDismiss` property):
       ```swift
       private var cursorAtTrigger: CGPoint = .zero
       private var resizeObserver: (any NSObjectProtocol)?
       ```

    2. **Refactor `activeScreen()` to accept a point** — rename to `screen(containing:)`:
       ```swift
       private func screen(containing point: CGPoint) -> NSScreen {
           if let screen = NSScreen.screens.first(where: { NSMouseInRect(point, $0.frame, false) }) {
               return screen
           }
           if let screen = NSScreen.main ?? NSScreen.screens.first {
               return screen
           }
           fatalError("PopupController requires at least one NSScreen")
       }
       ```

    3. **Add `repositionPanel()` method** (in the Screen placement MARK section):
       ```swift
       private func repositionPanel() {
           let screen = screen(containing: cursorAtTrigger)
           let origin = PopupPositionCalculator.calculateOrigin(
               cursorLocation: cursorAtTrigger,
               panelSize: panel.frame.size,
               screenFrame: screen.visibleFrame
           )
           panel.setFrameOrigin(origin)
       }
       ```

    4. **Modify `show()` method** — replace the positioning line (currently line 59):
       - After `panel.contentView = hostingView` and before `panel.alphaValue = 0`:
         ```swift
         // Remove old resize observer from previous show() (handles rapid re-trigger)
         removeResizeObserver()

         cursorAtTrigger = NSEvent.mouseLocation
         repositionPanel()

         resizeObserver = NotificationCenter.default.addObserver(
             forName: NSWindow.didResizeNotification,
             object: panel,
             queue: .main
         ) { [weak self] _ in
             MainActor.assumeIsolated {
                 self?.repositionPanel()
             }
         }
         ```
       - DELETE the old line: `panel.setFrameOrigin(topCenterOrigin(for: panel))`

    5. **Add `removeResizeObserver()` helper**:
       ```swift
       private func removeResizeObserver() {
           if let observer = resizeObserver {
               NotificationCenter.default.removeObserver(observer)
               resizeObserver = nil
           }
       }
       ```

    6. **Modify `dismiss()` method** — add resize observer cleanup before `panel.orderOut(nil)`:
       ```swift
       removeResizeObserver()
       ```

    7. **DELETE `topCenterOrigin(for:)` method entirely** (lines 117-125). It is fully replaced by `repositionPanel()` + `PopupPositionCalculator`.

    8. **Run `xcodegen generate`** to ensure Xcode project is up to date.

    **Top-anchored growth explanation:** When content height changes (loading text → translated result), `NSWindow.didResizeNotification` fires. `repositionPanel()` recalculates from `cursorAtTrigger`. Since `calculateOrigin` places the popup top edge at `cursorY - offset`, the top stays anchored and the popup grows downward. If new height causes overflow, edge-clamping/flip rules apply automatically.
