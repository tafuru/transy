Install xcodegen if not already present (`brew install xcodegen`), then create `project.yml` in the repo root with the following specification:

```yaml
name: Transy
options:
  bundleIdPrefix: com.tafuru
  deploymentTarget:
    macOS: "15.0"
  xcodeVersion: "16.0"
  indentWidth: 4
  tabWidth: 4

targets:
  Transy:
    type: application
    platform: macOS
    deploymentTarget: "15.0"
    sources:
      - path: Transy
    settings:
      base:
        SWIFT_VERSION: 6.0
        MACOSX_DEPLOYMENT_TARGET: "15.0"
        PRODUCT_BUNDLE_IDENTIFIER: com.tafuru.transy
        INFOPLIST_FILE: Transy/Info.plist
        CODE_SIGN_STYLE: Automatic
        ENABLE_APP_SANDBOX: NO
        CODE_SIGN_ENTITLEMENTS: ""
    info:
      path: Transy/Info.plist
      properties:
        CFBundleName: Transy
        CFBundleDisplayName: Transy
        CFBundleIdentifier: $(PRODUCT_BUNDLE_IDENTIFIER)
        CFBundleShortVersionString: "0.1.0"
        CFBundleVersion: "1"
        LSMinimumSystemVersion: "15.0"
        LSUIElement: true
        NSHumanReadableCopyright: ""

  TransyTests:
    type: bundle.unit-test
    platform: macOS
    deploymentTarget: "15.0"
    sources:
      - path: TransyTests
    settings:
      base:
        SWIFT_VERSION: 6.0
        MACOSX_DEPLOYMENT_TARGET: "15.0"
    dependencies:
      - target: Transy

  TransyUITests:
    type: bundle.ui-testing
    platform: macOS
    deploymentTarget: "15.0"
    sources:
      - path: TransyUITests
    settings:
      base:
        SWIFT_VERSION: 6.0
        MACOSX_DEPLOYMENT_TARGET: "15.0"
    dependencies:
      - target: Transy
```

Key configuration notes:
- `LSUIElement: true` — Info.plist Boolean YES, not an entitlement. This is the primary mechanism for no-Dock presence; runtime policy call is belt-and-suspenders only.
- `ENABLE_APP_SANDBOX: NO` and `CODE_SIGN_ENTITLEMENTS: ""` — no sandbox. Phase 2 requires `NSEvent.addGlobalMonitorForEvents` which is blocked by App Sandbox. Do NOT add a sandbox entitlement.
- `SWIFT_VERSION: 6.0` — strict concurrency enforcement from day one.
- `deploymentTarget: "15.0"` — required for `@Observable` (macOS 14+, using 15+ floor) and `openSettings` env action (macOS 14+).

After writing `project.yml`, create the source directories and run:
```bash
mkdir -p Transy/MenuBar Transy/Settings TransyTests TransyUITests
xcodegen generate
```
